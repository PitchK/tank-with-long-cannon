How to fold: Tank w/ long cannon

Time to fold: 8-13 min. | Difficulty:

5/10

Time to fold: 8-13 min.

Difficulty:

5/10

This model is simple yet fun to fold at the same time - its main feature is the long cannon that extends more than the body's length. Unlike my other tanks that looks like it can shoot only straight ahead, this looks as if it could lob a cannonball in a parabolic curve. Oh, and folding it on a 6in x 6in (15cm by 15cm) paper would be able to perfectly seat a Lego minifigure behind the cannon as well. This is one unique model!
